# Demo Mall
#### 项目介绍
```
构建项目用的是 create-react-app
状态管理用的是 mobx
UI框架用的是 antd（pc）、antd-mobile（h5）
http发送请求用的是 axios
后台模拟用的是 express
数据库用的是 mysql
```
#### 项目预览
```
路径：https://github.com/ZhongTaisong/create-react-app-mobx/tree/master/preview
```
#### 文件目录介绍
```
pc - pc前端代码
h5 - h5前端代码
server - 后台模拟接口代码
sql -数据库
preview（项目预览图片）
```
