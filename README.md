# Demo Mall
#### 项目介绍
```
构建项目用的是 create-react-app（pc、h5）、Taro（微信小程序）
状态管理用的是 mobx（pc、h5、微信小程序）
UI框架用的是 antd（pc）、antd-mobile（h5）、taro-ui（微信小程序）
http发送请求用的是 axios（pc、h5）、Taro.request(微信小程序)
后台模拟用的是 express
数据库用的是 mysql
```
